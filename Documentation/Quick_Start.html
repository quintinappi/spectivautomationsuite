📋 Before You Start

  • Autodesk Inventor is running
  • Your main assembly is open in Inventor
  • All part files are accessible (no read-only locks)
  • You've decided on a project prefix (PLANT1-000-, AREA2-000-, etc.)
⚠️ Recommended: Backup your project folder before running. While the heritage method preserves original files, a backup provides extra safety.

🎯 3-Step Process

1 Launch & Rename Parts
Double-click MAIN_LAUNCHER.bat → Select [1] Part Renaming
Enter your project prefix when prompted (e.g., PLANT1-000-)
2 Update IDW Drawings
Return to main menu → Select [2] IDW Updates
This automatically updates all your drawings to reference the new part names
3 Update IDW Drawings & Titles
Return to main menu → Select [2] IDW Updates
Optional: Select [3] Title Automation for automatic title formatting
✓ Complete!
Parts renamed, drawings updated, and titles formatted.
Check log files to verify everything completed successfully.

🎛️ What Each Step Does

Step 1: Part Renaming

  • Scans your entire assembly hierarchy
  • Groups parts automatically (PL=platework, B=beams, CH=channels, etc.)
  • Creates heritage copies with new names (preserves design relationships)
  • Updates assembly references to point to new parts
  • Saves mapping file for Step 2

Step 2: IDW Updates

  • Loads the mapping from Step 1
  • Finds all IDW drawing files in your project
  • Updates each drawing to reference the new part names
  • Uses Autodesk Design Assistant method for reliability
  • Saves all updated drawings

Step 3: Title Automation (Optional)

  • Detects base views in IDW drawings automatically
  • Generates titles based on assembly names and quantities
  • Formats titles with proper font sizes and layout
  • Updates title blocks for parts and assemblies
  • Processes multiple IDW files in batch

✅ How to Know It Worked

Success Indicators:

  • Both steps complete without error messages
  • Log files show "SUCCESS" entries
  • Your parts now have project prefix names (PLANT1-000-PL1.ipt instead of Part1.ipt)
  • IDW drawings open correctly and show the new part names
  • Assembly still functions normally with all constraints intact

🚨 Common Issues

Problem Quick Fix
"No running Inventor instance found" Start Inventor and open your assembly first
"Could not load mapping file" Run Step 1 (Part Renaming) first
"Access denied" errors Check file permissions, remove read-only locks
Parts not grouping correctly Set Description property in part iProperties

💡 Pro Tips

  • Test first: Try on a small assembly before large projects
  • Stay connected: Keep Inventor open throughout the entire process
  • Check logs: Always review log files after each step completes
  • Run in sequence: Always complete Step 1 before starting Step 2
  • Project prefixes: Use consistent naming like PLANT1-000-, AREA2-000-
