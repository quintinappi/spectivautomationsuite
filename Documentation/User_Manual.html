🚀 Quick Start

3-Step Process:

  1. Double-click MAIN_LAUNCHER.bat → Select [1] Part Renaming
  2. Return to main menu → Select [2] IDW Updates
  3. Done! All parts renamed and IDW drawings updated

📋 What You Need Before Starting

Requirement Status Check
Autodesk Inventor running ✅ Open with your main assembly loaded
All part files accessible ✅ No read-only locks or network issues
Project prefix decided ✅ Choose: PLANT1-000-, AREA2-000-, etc.
Backup (recommended) ⚠️ Copy project folder before running

🎯 Step-by-Step Instructions

Step 1: Launch Main Menu

Navigate to FINAL_PRODUCTION_SCRIPTS folder

Double-click: MAIN_LAUNCHER.bat

INVENTOR RENAMER - MAIN LAUNCHER === MAIN WORKFLOW === [1] Part Renaming - Rename assembly parts with heritage method [2] IDW Updates - Update IDW references to new part names [3] Title Automation - Auto-update IDW drawing titles === UTILITIES === [4] Registry Management - Manage numbering counters and database [5] File Utilities - Find duplicate files and conflicts === RESCUE TOOLS === [6] Smart Prefix Scanner - Scan model to prevent duplicate numbers [7] Emergency IDW Fixer - Fix specific folders missed by Step 2 [8] IDW-Assembly Synchronizer - Sync IDWs to assembly (scattered files) [9] Assembly Cloner - Clone assembly + parts to new folder [10] Part Cloner - Clone individual part to new folder === iLOGIC TOOLS === [11] iLogic Scanner - Scan/export iLogic rules from document [0] Exit

Step 2: Part Renaming

Select Option [1] - Part Renaming

Prefix Input:

Enter your project prefix when prompted:

  • PLANT1-000- (Plant 1)
  • AREA2-000- (Area 2)
  • NCRH01-000- (default)
  • Or press Enter for default

What Happens:

  1. Scans your entire assembly hierarchy
  2. Groups parts automatically:
    • PL = Platework ("PL 20mm S355JR")
    • B = Beams/Columns ("UB254x146x31")
    • CH = Channels ("PFC180x70")
    • A = Angles ("L50x50x6")
    • FL = Flatbar ("FL50x8")
  3. Creates heritage copies with new names
  4. Updates assembly references
  5. Saves mapping file for Step 2
⚠️ Important: Keep Inventor open during processing! Don't close or switch projects.

Step 3: IDW Updates

Return to main menu, select Option [2] - IDW Updates

What Happens:

  1. Loads mapping from Step 1
  2. Finds all IDW drawing files
  3. Updates each drawing to reference new part names
  4. Uses Autodesk Design Assistant method
  5. Saves all updated drawings

Results:

Your IDW drawings now open with the new part names (PLANT1-000-PL1.ipt instead of Part1.ipt)

Step 4: Title Automation (Optional)

Return to main menu, select Option [3] - Title Automation

What It Does:

  • Automatically detects base views in IDW drawings
  • Updates title blocks with proper formatting
  • Generates titles based on assembly names and quantities
  • Applies correct font sizes and formatting

Title Format Examples:

Part Type Title Format
Individual Parts PLANT1-000-PL1
SCALE 1:5
Assemblies PLANT1-000-BA1
7-OFF REQ'D
SCALE 1:20

Prerequisites:

  • Steps 1 and 2 completed successfully
  • IDW files accessible
  • Inventor still running

Step 5: Verify Results

✅ Success Indicators:

  • All steps complete without errors
  • Log files show "SUCCESS" messages
  • IDW files open correctly with new part references
  • Parts now have project prefix names
  • Drawing titles are properly formatted (if Step 4 completed)

Assembly Cloner (Rescue Tool)

Select Option [9] - Assembly Cloner

What It Does:

  • Copies entire assembly + all parts to a new isolated folder
  • Updates all references to use local copies
  • Optionally applies heritage renaming (PL, CH, B, etc.)
  • Copies and updates associated IDW drawings
  • Creates fully isolated clone with no cross-references

When to Use:

  • Creating variants without cross-references
  • Copying assemblies to new projects
  • Isolating assemblies for modification
  • Fixing IDW reference mismatches
⚠️ Important: Ensure IDW references match assembly parts for successful updates. Sync IDW model references if needed before cloning.

Part Cloner (Rescue Tool)

Select Option [10] - Part Cloner

What It Does:

  • Copies individual part (.ipt) to a new isolated folder
  • Displays part iProperties (title, author, description, etc.)
  • Optionally renames the part file
  • Creates isolated copy for modification

When to Use:

  • Creating part variants
  • Isolating parts for modification
  • Backup copies of individual parts
  • Reviewing part properties before cloning
⚠️ Important: Open the source part in Inventor before running. No assembly or IDW complications.

🔧 Additional Tools

Registry Management

Select Option [4] - Registry Management

SCAN Function:

  • Enter prefix to check current numbering state
  • Shows: PLANT1-000-PL = 15, PLANT1-000-B = 8, etc.

CLEAR Function:

  • ⚠️ DANGEROUS: Resets all counters to zero
  • Next run starts fresh (PL1, B1, CH1, etc.)
  • Also deletes mapping file
  • Requires double confirmation

File Utilities

Select Option [5] - File Utilities

Scans for duplicate files that might cause issues. Run before Part Renaming if you suspect naming conflicts.

🚨 Troubleshooting

Problem Solution
"No running Inventor instance found" Start Inventor and open your assembly
"Could not load mapping file" Run Step 1 (Part Renaming) first
"Access denied" errors Check file permissions, remove read-only attributes
IDW files won't open Check log files for specific errors
Parts not grouping correctly Set Description property in part iProperties

💡 Pro Tips

  • Test first: Try on a small assembly before large projects
  • Check logs: Always review log files after each step
  • Backup project: Copy entire project folder before running
  • Sequential processing: Always run Step 1 before Step 2
  • Keep Inventor open: Don't close during processing
  • Sync IDW references: If IDW updates fail, ensure drawing model references match current assembly parts
  • Use Assembly Cloner: For creating isolated copies or fixing reference issues
  • Mapping files: Now saved per-assembly in the assembly folder for safety
