● I've completed the implementation of the Beam Assembly Generator for your Inventor 2026 add-in. Here's a summary of what was created:

  Files Created

  1. SteelSectionData.vb - SANS Steel Section Database

  Contains comprehensive steel section data including:
  - Universal Columns (UC): 152x152, 203x203, 254x254, 305x305, 356x368 series
  - Universal Beams (UB): 127x76 through 610x305 series
  - Parallel Flange Channels (PFC): 100x50 through 400x100
  - Tapered Flange Channels (TFC): 76x38 through 381x102
  - Equal Angles (L): 25x25 through 200x200
  - Circular Hollow Sections (CHS): 33.7 through 168.3 diameter
  - Square Hollow Sections (SHS): 25x25 through 200x200
  - Flat Bar (FL): Various widths and thicknesses

  Each section includes: height, width, web thickness, flange thickness, mass, backmark distances, hole count, bolt size, and hole diameter.

  2. BeamAssemblyGenerator.vb - Core Generator Logic

  Creates parametric beam assemblies with:
  - Beam profile sketches: I-beam, channel, and angle profiles drawn dynamically
  - Endplates: Sized to match beam section with proper overhang
  - Hole patterns:
    - 4 holes for UC/UB (2x2 pattern at correct backmarks)
    - 2 holes for PFC/TFC (single row in web)
    - 2 holes for angles (one in each leg)
  - Hole sizes: 18mm for M16, 22mm for M20, 26mm for M24
  - Assembly constraints: Endplates positioned at beam ends
  - iProperties: Automatically set (e.g., "PL 12mm S355JR" for endplates, "UC 203 x 203 x 46" for beams)

  3. BeamGeneratorForm.vb - User Interface

  Windows Forms dialog with:
  - Section Type dropdown: UC, UB, PFC, TFC, L, CHS, SHS, RHS, FL
  - Section Size dropdown: Populated dynamically based on type
  - Section Properties display: Shows dimensions, mass, hole count, backmarks
  - Beam Length input: In millimeters
  - Part Number Prefix: Default "PLANT-000-"
  - Output Folder browser
  - Preview panel: Shows what files will be created

  4. Updated StandardAddInServer.vb

  Added:
  - New "Beam Generator" button with teal icon (B)
  - Button appears in both Assembly and Part ribbons
  - Handler launches the BeamGeneratorForm

  5. Updated AssemblyClonerAddIn.vbproj

  Added the three new source files to the project.

  2. Build the project (Ctrl+Shift+B)
  3. Copy the output DLL and .addin file to Inventor's Add-Ins folder:
  C:\ProgramData\Autodesk\Inventor Addins\
  4. Start Inventor 2026
  5. Click "Beam Generator" button in the Tools ribbon
  6. Select section type/size, enter beam length, choose output folder
  7. Click "Create Assembly"

  Key Features

  | Section Type   | Hole Count | Hole Size  | Backmark Logic           |
  |----------------|------------|------------|--------------------------|
  | UC/UB (wide)   | 4          | 22mm (M20) | From center, 2x2 pattern |

        92 +          Private Sub CreateBeamAssembly()
        93 +              ' Ensure output folder exists
        94 +              If Not 
           + System.IO.Directory.Exists(m_OutputFolder) Then
        95 +                  
           + System.IO.Directory.CreateDirectory(m_OutputFolder)
        96 +              End If
        97 +  
        98 +              ' Generate file names
        99 +              Dim sectionCode As String = 
           + m_CurrentSection.SectionType
       100 +              Dim timestamp As String = 
           + DateTime.Now.ToString("HHmmss")
       101 +              Dim baseName As String = m_PartNumberPrefix & 
           + sectionCode & "-" & timestamp
       102 +  
       103 +              m_BeamPartPath = 
           + System.IO.Path.Combine(m_OutputFolder, baseName & 
           + "-BEAM.ipt")
       104 +              m_EndplatePath = System.IO.
           +  Path.Combine(m_OutputFolder, baseName & "-ENDPLATE.ipt")
       105 +              m_AssemblyPath = System.IO.
           +  Path.Combine(m_OutputFolder, baseName & ".iam")
       106 +  
       107 +              ' Step 1: Create the beam part
       108 +              CreateBeamPart()
       109 +  
       110 +              ' Step 2: Create the endplate part
       111 +              CreateEndplatePart()
       112 +  
       113 +              ' Step 3: Create the assembly with constraints

       119 +          ''' <summary>
       120 +          ''' Create the beam part with extruded profile
       121 +          ''' </summary>
       122 +          Private Sub CreateBeamPart()
       123 +              ' Create new part document
       124 +              Dim partDoc As PartDocument = CType(m_InventorA
           + pp.Documents.Add(DocumentTypeEnum.kPartDocumentObject, , 
           + True), PartDocument)
       125 +              Dim compDef As PartComponentDefinition = 
           + partDoc.ComponentDefinition
       126 +  
       127 +              ' Create sketch on XY plane (front view)
       128 +              Dim sketch As PlanarSketch = 
           + compDef.Sketches.Add(compDef.WorkPlanes.Item(3)) ' XY plane
       129 +  
       130 +              ' Draw the profile based on section type
       131 +              Select Case m_CurrentSection.SectionType
       132 +                  Case "UC", "UB"
       133 +                      DrawIBeamProfile(sketch)
       134 +                  Case "PFC", "TFC"
       135 +                      DrawChannelProfile(sketch)
       136 +                  Case "L"
       137 +                      DrawAngleProfile(sketch)
       138 +                  Case Else
       139 +                      DrawIBeamProfile(sketch) ' Default to 
           + I-beam
       140 +              End Select
       141 +  
       142 +              ' Create profile and extrude
       143 +              Dim profile As Profile = 
           + sketch.Profiles.AddForSolid()
       144 +  
       145 +              ' Convert beam length from mm to cm
       146 +              Dim lengthCm As Double = m_BeamLength / 10.0
       147 +  
       148 +              ' Create symmetric extrusion using 
           + AddByDistanceExtent
       149 +              Dim extrude As ExtrudeFeature = 
           + compDef.Features.ExtrudeFeatures.AddByDistanceExtent(
       150 +                  profile,
       151 +                  lengthCm,
       152 +                  
           + PartFeatureExtentDirectionEnum.kSymmetricExtentDirection,
       153 +                  PartFeatureOperationEnum.kNewBodyOperation)
       154 +  
       155 +              ' Set iProperties
       156 +              SetPartIProperties(partDoc, "BEAM", 
           +  m_CurrentSection.GetIPropertyDescription())
       157 +  
       158 +              ' Save the beam part
       159 +              partDoc.SaveAs(m_BeamPartPath, False)
       160 +              partDoc.Close(False)
       161 +          End Sub

       163 +          ''' <summary>
       164 +          ''' Draw I-beam/H-section profile (for UC and UB)
       165 +          ''' </summary>
       166 +          Private Sub DrawIBeamProfile(sketch As 
           +  PlanarSketch)
       167 +              Dim lines As SketchLines = sketch.SketchLines
       168 +  
       169 +              ' Get dimensions in cm (Inventor's internal 
           +  unit)
       170 +              Dim h As Double = m_CurrentSection.Height / 
           +  10.0    ' Height
       171 +              Dim w As Double = m_CurrentSection.Width / 10.0
           +        ' Flange width
       172 +              Dim tw As Double = 
           +  m_CurrentSection.WebThickness / 10.0  ' Web thickness
       173 +              Dim tf As Double = 
           +  m_CurrentSection.FlangeThickness / 10.0  ' Flange thickness
       174 +  
       175 +              ' Half dimensions for symmetric drawing
       176 +              Dim hh As Double = h / 2.0   ' Half height
       177 +              Dim hw As Double = w / 2.0   ' Half width
       178 +              Dim htw As Double = tw / 2.0 ' Half web 
           +  thickness
       179 +  
       180 +              ' Draw I-beam profile (centered at origin)
       181 +              ' Top flange - outside
       182 +              Dim p1 As Point2d = 
           +  m_TransientGeometry.CreatePoint2d(-hw, hh)
       183 +              Dim p2 As Point2d = 
           +  m_TransientGeometry.CreatePoint2d(hw, hh)
       184 +              Dim p3 As Point2d = 
           +  m_TransientGeometry.CreatePoint2d(hw, hh - tf)
       185 +              Dim p4 As Point2d = 
           +  m_TransientGeometry.CreatePoint2d(htw, hh - tf)
       186 +  
       187 +              ' Web - right side
       188 +              Dim p5 As Point2d = 
           +  m_TransientGeometry.CreatePoint2d(htw, -hh + tf)
       189 +  
       190 +              ' Bottom flange - right side
       191 +              Dim p6 As Point2d = 
           +  m_TransientGeometry.CreatePoint2d(hw, -hh + tf)
       192 +              Dim p7 As Point2d = 
           +  m_TransientGeometry.CreatePoint2d(hw, -hh)
       193 +              Dim p8 As Point2d = 
           +  m_TransientGeometry.CreatePoint2d(-hw, -hh)
       194 +              Dim p9 As Point2d = 
           +  m_TransientGeometry.CreatePoint2d(-hw, -hh + tf)
       195 +              Dim p10 As Point2d = 
           +  m_TransientGeometry.CreatePoint2d(-htw, -hh + tf)
       196 +  
       197 +              ' Web - left side
       198 +              Dim p11 As Point2d = 
           +  m_TransientGeometry.CreatePoint2d(-htw, hh - tf)
       199 +  
       200 +              ' Top flange - left side
       201 +              Dim p12 As Point2d = 
           +  m_TransientGeometry.CreatePoint2d(-hw, hh - tf)
       202 +  
       203 +              ' Draw lines to form closed profile
       204 +              lines.AddByTwoPoints(p1, p2)
       205 +              lines.AddByTwoPoints(p2, p3)
       206 +              lines.AddByTwoPoints(p3, p4)
       207 +              lines.AddByTwoPoints(p4, p5)
       208 +              lines.AddByTwoPoints(p5, p6)
       209 +              lines.AddByTwoPoints(p6, p7)
       210 +              lines.AddByTwoPoints(p7, p8)
       211 +              lines.AddByTwoPoints(p8, p9)
       212 +              lines.AddByTwoPoints(p9, p10)
       213 +              lines.AddByTwoPoints(p10, p11)
       214 +              lines.AddByTwoPoints(p11, p12)
       215 +              lines.AddByTwoPoints(p12, p1)
       216 +          End Sub

       285 +          ''' <summary>
       286 +          ''' Create the endplate part with holes
       287 +          ''' </summary>
       288 +          Private Sub CreateEndplatePart()
       289 +              ' Create new part document
       290 +              Dim partDoc As PartDocument = CType(m_InventorA
           +  pp.Documents.Add(DocumentTypeEnum.kPartDocumentObject, , 
           +  True), PartDocument)
       291 +              Dim compDef As PartComponentDefinition = 
           +  partDoc.ComponentDefinition
       292 +  
       293 +              ' Determine endplate dimensions based on 
           +  section
       294 +              Dim plateWidth As Double = 
           +  m_CurrentSection.Width + 20 ' Section width + 10mm each 
           +  side
       295 +              Dim plateHeight As Double = 
           +  m_CurrentSection.Height + 20 ' Section height + 10mm each 
           +  side
       296 +              Dim plateThickness As Double = 12 ' Default 
           +  12mm endplate
       297 +  
       298 +              ' Adjust for specific section types
       299 +              Select Case m_CurrentSection.SectionType
       300 +                  Case "UC", "UB"
       301 +                      ' Endplate slightly larger than flange
       302 +                      plateWidth = m_CurrentSection.Width + 
           +  20
       303 +                      plateHeight = m_CurrentSection.Height +
           +   20
       304 +                      plateThickness = 
           +  If(m_CurrentSection.Height > 300, 20, 12)
       305 +                  Case "PFC", "TFC"
       306 +                      ' For channels, width is the height of 
           +  channel, height is the flange width
       307 +                      plateWidth = m_CurrentSection.Height + 
           +  20
       308 +                      plateHeight = m_CurrentSection.Width + 
           +  20
       309 +                      plateThickness = 10
       310 +                  Case "L"
       311 +                      ' For angles, plate based on leg sizes
       312 +                      plateWidth = m_CurrentSection.LegA + 20
       313 +                      plateHeight = m_CurrentSection.LegB + 
           +  20
       314 +                      plateThickness = 10
       315 +              End Select
       316 +  
       317 +              ' Create sketch on XY plane
       318 +              Dim sketch As PlanarSketch = 
           +  compDef.Sketches.Add(compDef.WorkPlanes.Item(3))
       319 +              Dim lines As SketchLines = sketch.SketchLines
       320 +  
       321 +              ' Convert to cm
       322 +              Dim pwCm As Double = plateWidth / 10.0
       323 +              Dim phCm As Double = plateHeight / 10.0
       324 +              Dim hpw As Double = pwCm / 2.0
       325 +              Dim hph As Double = phCm / 2.0
       326 +  
       327 +              ' Draw rectangle for endplate (centered)
       328 +              Dim p1 As Point2d = 
           +  m_TransientGeometry.CreatePoint2d(-hpw, -hph)
       329 +              Dim p2 As Point2d = 
           +  m_TransientGeometry.CreatePoint2d(hpw, -hph)
       330 +              Dim p3 As Point2d = 
           +  m_TransientGeometry.CreatePoint2d(hpw, hph)
       331 +              Dim p4 As Point2d = 
           +  m_TransientGeometry.CreatePoint2d(-hpw, hph)
       332 +  
       333 +              lines.AddByTwoPoints(p1, p2)
       334 +              lines.AddByTwoPoints(p2, p3)
       335 +              lines.AddByTwoPoints(p3, p4)
       336 +              lines.AddByTwoPoints(p4, p1)
       337 +  
       338 +              ' Create profile and extrude
       339 +              Dim profile As Profile = 
           +  sketch.Profiles.AddForSolid()
       340 +              Dim thicknessCm As Double = plateThickness / 
           +  10.0
       341 +  
       342 +              ' Create extrusion using AddByDistanceExtent
       343 +              Dim extrude As ExtrudeFeature = 
           + compDef.Features.ExtrudeFeatures.AddByDistanceExtent(
       344 +                  profile,
       345 +                  thicknessCm,
       346 +                  
           + PartFeatureExtentDirectionEnum.kPositiveExtentDirection,
       347 +                  PartFeatureOperationEnum.kNewBodyOperation)
       348 +  
       349 +              ' Add holes based on section type and backmarks
       350 +              AddEndplateHoles(compDef, plateWidth,

       351 +  
       352 +              ' Set iProperties
       353 +              Dim description As String = "PL " &

       356 +              ' Save
       357 +              partDoc.SaveAs(m_EndplatePath, False)
       358 +              partDoc.Close(False)

       361 +          ''' <summary>
       362 +          ''' Add holes to the endplate based on section 
           + backmarks
       363 +          ''' </summary>
       364 +          Private Sub AddEndplateHoles(compDef As 
           + PartComponentDefinition, plateWidth As Double, plateHeight 
           + As Double, plateThickness As Double)
       365 +              ' Get hole parameters
       366 +              Dim holeDiameter As Double = m_CurrentSection.
           +  HoleDiameter ' mm
       367 +              Dim holeCount As Integer = 
           + m_CurrentSection.HoleCount
       368 +              Dim backmarkX As Double = 
           + m_CurrentSection.BackmarkX ' mm from center
       369 +              Dim backmarkY As Double = 
           + m_CurrentSection.BackmarkY ' mm from center
       370 +  
       371 +              If holeCount = 0 Then Return ' No holes for 
           + hollow sections
       372 +  
       373 +              ' Create sketch for hole centers on front face
       374 +              Dim frontFace As Face = GetFrontFace(compDef)
       375 +              If frontFace Is Nothing Then Return
       376 +  
       377 +              Dim holeSketch As PlanarSketch = 
           + compDef.Sketches.Add(frontFace)
       378 +  
       379 +              ' Convert to cm
       380 +              Dim holeDiameterCm As Double = holeDiameter
           +   / 10.0
       381 +              Dim backmarkXCm As Double = backmarkX / 10.0
       382 +              Dim backmarkYCm As Double = backmarkY / 10.0
       383 +  
       384 +              ' Collect hole center points
       385 +              Dim holePoints As New 
           + System.Collections.Generic.List(Of Point2d)
       386 +  
       387 +              ' Add hole center points based on section type
       388 +              Select Case m_CurrentSection.SectionType
       389 +                  Case "UC", "UB"
       390 +                      ' 4 holes for I/H sections (2x2 
           + pattern)
       391 +                      If holeCount = 4 Then
       392 +                          holePoints.Add(m_TransientGeometry.
           + CreatePoint2d(backmarkXCm, backmarkYCm))
       393 +                          holePoints.Add(m_TransientGeometry.
           + CreatePoint2d(-backmarkXCm, backmarkYCm))
       394 +                          holePoints.Add(m_TransientGeometry.
           + CreatePoint2d(backmarkXCm, -backmarkYCm))
       395 +                          holePoints.Add(m_TransientGeometry.
           + CreatePoint2d(-backmarkXCm, -backmarkYCm))
       396 +                      ElseIf holeCount = 2 Then
       397 +                          ' Narrow sections with 2 holes 
           + (single row)
       398 +                          
           + holePoints.Add(m_TransientGeometry.CreatePoint2d(0, 
           + backmarkYCm))
       399 +                          
           + holePoints.Add(m_TransientGeometry.CreatePoint2d(0, 
           + -backmarkYCm))
       400 +                      End If
       401 +  
       402 +                  Case "PFC", "TFC"
       403 +                      ' 2 holes for channels (single row in 
           + web)
       404 +                      Dim channelBackmark As Double = 
           + backmarkXCm
       405 +                      
           + holePoints.Add(m_TransientGeometry.CreatePoint2d(0, 
           + channelBackmark))
       406 +                      
           + holePoints.Add(m_TransientGeometry.CreatePoint2d(0, 
           + -channelBackmark))
       407 +  
       408 +                  Case "L"
       409 +                      ' 2 holes for angles (one in each leg)
       410 +                      holePoints.Add(m_TransientGeometry.Crea
           + tePoint2d(backmarkXCm, 0))
       411 +                      
           + holePoints.Add(m_TransientGeometry.CreatePoint2d(0, 
           + backmarkYCm))
       412 +              End Select
       413 +  
       414 +              ' Create holes for each point using sketch 
           + circles and cut extrude
       415 +              For Each holeCenter As Point2d In holePoints
       416 +                  Try
       417 +                      ' Add sketch point for hole center
       418 +                      Dim sketchPt As SketchPoint = 
           + holeSketch.SketchPoints.Add(holeCenter, False)
       419 +  
       420 +                      ' Draw circle for hole
       421 +                      Dim circle As SketchCircle = 
           + holeSketch.SketchCircles.AddByCenterRadius(holeCenter, 
           + holeDiameterCm / 2.0)
       422 +                  Catch ex As Exception
       423 +                      ' Continue with next hole if this one 
           + fails
       424 +                  End Try
       425 +              Next
       426 +  
       427 +              ' Create cut extrude for holes if we have 
           + circles
       428 +              If holeSketch.SketchCircles.Count > 0 Then
       429 +                  Try
       430 +                      Dim holeProfile As Profile = 
           + holeSketch.Profiles.AddForSolid()
       431 +                      Dim holeCut As ExtrudeFeature = 
           + compDef.Features.ExtrudeFeatures.AddByDistanceExtent(
       432 +                          holeProfile,
       433 +                          plateThickness / 10.0 + 0.1, ' 
           + Extra depth to ensure through hole
       434 +                          
           + PartFeatureExtentDirectionEnum.kNegativeExtentDirection,
       435 +                          
           + PartFeatureOperationEnum.kCutOperation)
       436 +                  Catch ex As Exception
       437 +                      ' Holes failed but endplate is still 
           + valid
       438 +                  End Try
       439 +              End If
       440 +          End Sub

       442 +          ''' <summary>
       443 +          ''' Get the front face of the extruded plate (face 
           + with normal in Z direction)
       444 +          ''' </summary>
       445 +          Private Function GetFrontFace(compDef As 
           + PartComponentDefinition) As Face
       446 +              For Each body As SurfaceBody In 
           + compDef.SurfaceBodies
       447 +                  For Each face As Face In body.Faces
       448 +                      Try
       449 +                          Dim evalObj As Object = 
           + face.Geometry
       450 +                          If TypeOf evalObj Is Plane Then
       451 +                              Dim planeGeom As Plane = 
           + CType(evalObj, Plane)
       452 +                              If Math.Abs(planeGeom.Normal.Z)
           +  > 0.9 AndAlso planeGeom.Normal.Z > 0 Then
       453 +                                  Return face
       454 +                              End If
       455 +                          End If
       456 +                      Catch
       457 +                      End Try
       458 +                  Next
       459 +              Next
       460 +              Return Nothing
       461 +          End Function
