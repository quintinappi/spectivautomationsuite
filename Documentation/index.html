🎯 What Is Inventor Renamer?

A comprehensive solution for systematically renaming Autodesk Inventor assembly parts with intelligent classification and automatic IDW drawing reference updates.

  • Heritage-Based Method: Preserves design relationships and file history
  • Intelligent Classification: Automatically groups parts (PL, B, CH, A, etc.)
  • Dynamic Prefixes: Support for any project naming convention
  • Complete Workflow: From part renaming to IDW updates in 2 steps
  • Production Ready: Thoroughly tested and professionally organized

🚀 Getting Started

New to Inventor Renamer? Start here:

  1. Read the Quick Start Guide for immediate setup
  2. Open Inventor with your assembly
  3. Double-click MAIN_LAUNCHER.bat to begin
  4. Follow the process: Part Renaming → IDW Updates → Title Automation
